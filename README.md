# ЛАЗЕР · 7 — WordPress-тема (ACF Pro)

Двуязычный (UA / EN) одностраничный лендинг мастерской лазерной резки и
гравировки «ЛАЗЕР · 7» (Одесса, 7 км). Дизайн перенесён 1:1. **Абсолютно весь
контент** — тексты, заголовки, описания, фото, видео, цены, контакты, шапка и
подвал — редактируется через **Advanced Custom Fields PRO**: в настройках темы и
в настройках страницы.

---

## Требования

- WordPress 6.0+
- PHP 7.4+
- Плагин **Advanced Custom Fields PRO** (обязательно — используются Repeater,
  Options Page, nested-репитеры).

## Установка

1. Установите и активируйте плагин **ACF Pro**.
2. Скопируйте папку `laser7` в `wp-content/themes/` (или загрузите ZIP через
   «Внешний вид → Темы → Добавить новую → Загрузить тему»).
3. Активируйте тему **«ЛАЗЕР · 7»**.
4. При активации автоматически:
   - создаётся страница **«Головна»** и назначается главной;
   - в медиабиблиотеку импортируются изображения (webp);
   - **все поля заполняются демо-контентом** из дизайна — сайт сразу выглядит
     1:1, и всё уже готово к редактированию.
   > Заполнение происходит на первом заходе в админку после активации (когда
   > ACF готов). Если что-то не заполнилось — просто обновите любую страницу
   > админки один раз.

## Где что редактировать

### 🎛 Настройки темы (глобальное)
Меню в админке: **«ЛАЗЕР · 7»** (иконка-супергерой). Вкладки:
- **Бренд / Шапка** — акцентный цвет, верхняя панель (адрес, график, телефон),
  название бренда, пункты меню.
- **Контакты / Месенджери** — телефон, e-mail и карточки Telegram / Viber /
  WhatsApp (используются в шапке, секции «Контакти», подвале и плавающей кнопке).
- **Подвал** — слоган, копирайт, заголовки колонок, доп. ссылки.

### 📄 Настройки страницы (контент секций)
Откройте на редактирование страницу **«Головна»** — ниже редактора будет блок
**«ЛАЗЕР · 7 — контент сторінки»** с вкладками по секциям: Hero, Переваги, Про
майстерню, Портфоліо, Прайс, Матеріали, Відгук, FAQ, Локація, Контакти.

### 🌍 Двуязычность (UA / EN)
Каждое текстовое поле имеет два варианта: **· UA** и **· EN**. Переключатель
языка в шапке (UA / EN) мгновенно меняет язык (выбор запоминается в браузере).
Фото, видео, цены, ссылки — общие для обоих языков.

### 🖼 Портфолио
Вкладка **«Портфоліо»** → репитер **«Роботи»**: для каждой работы — фото
(или видео), название (UA/EN), материал (UA/EN), категория (ID из списка
категорий выше) и «тон» плейсхолдера. Категории-фильтры редактируются там же.
Фильтр, пагинация («Показати ще») и лайтбокс работают автоматически.

## Изображения

Дизайн ссылался на тяжёлые JPG/PNG — в теме используются сжатые **`.webp`** с
теми же именами (`assets/images/`). Любое фото можно заменить через поле ACF
(медиабиблиотека).

> ⚠️ Для одной работы — «Картхолдер зі шкіри» — webp-версии в исходниках не было
> (`leather-wallet.webp` отсутствовал), поэтому временно используется
> `leather-wallet.jpg`. Замените её на webp через ACF, когда будет готова.

## Акцентный цвет

По умолчанию — фирменный красный `#e0392b` (как в дизайне). В «Настройки темы →
Бренд / Шапка → Акцентний колір» можно выбрать: красный / янтарь / синий /
зелёный / графит.

## Структура темы

```
laser7/
├── style.css                 заголовок темы
├── functions.php             настройка, подключение, опции ACF, авто-страница
├── front-page.php            сборка лендинга
├── header.php / footer.php   шапка (топбар+меню+мобильное меню) / подвал
├── index.php                 запасной шаблон
├── assets/
│   ├── css/styles.css        дизайн 1:1 (из хэндофф-бандла)
│   ├── css/theme.css         двуязычность + мелкие правки
│   ├── js/main.js            язык, меню, FAQ, фильтр+лайтбокс, кнопки
│   └── images/               сжатые webp
├── inc/
│   ├── helpers.php           двуязычный вывод, изображения, опции
│   ├── icons.php             инлайн-SVG иконки и мотивы
│   ├── defaults.php          весь дефолтный контент (UA/EN)
│   ├── acf-fields.php        регистрация всех полей ACF
│   └── seed.php              разовое заполнение контентом + импорт фото
├── template-parts/sections/  шаблоны секций
└── acf-json/                 ACF сохраняет сюда определения групп
```

## Технические заметки

- Если нужно **повторно заполнить** демо-контент: удалите опцию
  `laser7_needs_seed`… проще — в «Настройки → Чтение» убедитесь, что главная =
  «Головна», и при необходимости заполните поля вручную (демо уже на месте).
- Тема не зависит от языка сайта в WP — переключатель UA/EN чисто фронтовый
  (как в исходном прототипе), контент обоих языков хранится в ACF.
- WebP-загрузки разрешаются темой (`upload_mimes`).
